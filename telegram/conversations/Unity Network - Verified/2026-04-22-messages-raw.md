Jaba - Unityfarmers, [Apr 22, 2026 at 11:15]
Additional procedure to extract 1,000 records from the manage.unitynodes.io page (after the website update it extracts only the first 20 by default)

1. Open DevTools (F12) → Go to Network tab.
2. Reload the page and find the request named rewards_get_allocations.
3. Right-click on that request row → Copy → Copy as fetch.
4. Go to Console tab.
5. Paste the code.
6. If Chrome says “Paste is not allowed”, type allow pasting, press Enter and paste the text again.
7. Make these two changes only:
    Change "take": 20 → "take": 1000
    Change "credentials": "include" → "credentials": "same-origin"

8. Press Enter.
9. Wait a few seconds → Move back to Network tab and copy the results that include 1,000 records from the  rewards_get_allocations which is bigger in size, look for 90kb or so
