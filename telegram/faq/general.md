## "What's a UNO vs ULO?" — 💬💬

(Background, not currently asked but useful to note for new members.)
- **UNO** = Unity Node Operator. Owns license(s). Bought from Unity Nodes for $25 each.
- **ULO** = Unity License Operator. Runs the Unity app on a phone, performs tasks, earns rewards. Leases the license from a UNO via marketplace or direct code.
- The two split rewards according to the lease split (default 50/50; UNO sets the percentage).
