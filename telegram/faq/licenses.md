## "What is a 'bound' license vs a leased one?" — 💬💬

A bound license = a license linked to a device (running). A leased license = made available to a ULO via marketplace or code. The web panel shows three counts: Total / Leased / Bound. Filters between these states are limited (see MikiRoma87 [Apr 27] complaint in feature-requests).

## "Can a license run without a node? Can I receive 100% of rewards as a standalone license operator?" — 💬💬 (clarified Apr 28)

**Asked by:** Wattdog [Apr 28], Akira [Apr 28], Tap MastersTaps [Apr 28].

**Community-derived answer (consensus from Wattdog, Alex, Calus B, Rick Middleton, Gwims, Apr 28):**
- A license is its own NFT, separate from the node NFT.
- **Yes — a standalone license can run without a node.** You just hold the license NFT in a web3 wallet (e.g. MetaMask) and log in as ULO. You then earn 100% of rewards.
- **You cannot lease the license out without a node.** Leasing requires a node. So a standalone license is "yours, you run it, 100%" — no split.
- A node is what allows leasing. A node with 0 licenses still has value (the leasing capacity itself).
- Gwims [Apr 28] confirms running this configuration since December 2025.

**Team answer:** [NEEDS OFFICIAL CONFIRMATION] — community consensus is consistent but not officially restated.

## "What does a 'lease never activated' look like and how do I revoke them in bulk?" — 💬💬 (NEW Apr 29)

**Asked by:** Rocky NOW pool [Apr 29].

Sample ask:
- Rocky NOW pool [Apr 29]: "I have generated 2000 licenses and handed them out. At this point I have to manually revoke the ones that haven't held any uptime or next to nothing. I have no available leases left to generate."

**Community answer (Jaba, Calus B, Apr 29):**
- A lease that's been generated but never activated on a device shows a **grey "phone rectangle" icon** (vs the illuminated paper/phone icons for activated leases). See Jaba's screenshot in the Apr 29 conversation file.
- If you revoke a lease and re-generate, you get a **different lease code but the same license ID**.
- Workaround until stackable filters ship: use the NexusNerd Chrome extension's snapshot dashboard (filter offline + zero 7-day rewards → bulk retract). Or implicit-date-stamp by bumping configured uptime % by 1% every 200 codes you hand out (Rocky's trick).
